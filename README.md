
# Smart Lease Management Portal - Backend API

ASP.NET Core Web API backend for Smart Lease Management Portal.

## Technologies

* C#
* ASP.NET Core Web API
* Entity Framework Core
* SQL Server
* Swagger

## Features

* Properties Management API
* Tenants Management API
* Leases Management API
* CRUD Operations
* SQL Server Integration
* Entity Framework Core Migrations
* Swagger API Documentation

## Architecture

React Frontend → ASP.NET Core Web API → Entity Framework Core → SQL Server

## Database

LeaseManagementDB

## API Endpoints

* GET /api/Properties

* POST /api/Properties

* PUT /api/Properties/{id}

* DELETE /api/Properties/{id}

* GET /api/Tenants

* POST /api/Tenants

* PUT /api/Tenants/{id}

* DELETE /api/Tenants/{id}

* GET /api/Leases

* POST /api/Leases

* PUT /api/Leases/{id}

* DELETE /api/Leases/{id}
